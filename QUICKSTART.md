# Быстрый старт

## Самый простой способ запуска:

1. **Откройте терминал** в папке проекта `CV-master`

2. **Выполните команды по порядку:**

```bash
# Установка зависимостей (первый раз)
pip install -r requirements.txt

# Запуск сервиса
uvicorn main:app --reload
```

3. **Откройте браузер** и перейдите на:
   ```
   http://localhost:8000
   ```

4. **Готово!** Теперь вы можете загружать резюме и анализировать вакансии.

---

## Альтернативный способ (через скрипт):

```bash
# Сделать скрипт исполняемым
chmod +x run.sh

# Запустить
./run.sh
```

---

## Остановка сервиса:

Нажмите `Ctrl + C` в терминале

---

## Если что-то не работает:

- Убедитесь, что Python установлен: `python --version` (должно быть 3.8+)
- Если команда `uvicorn` не найдена, попробуйте: `python -m uvicorn main:app --reload`
- Если порт 8000 занят, используйте другой: `uvicorn main:app --reload --port 8001`
